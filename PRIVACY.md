# Privacy Policy — Credentials Changer

**Last updated:** March 2026

## Data Collection

Credentials Changer does not collect, transmit, or share any user data.

All cookie values saved through the extension are stored locally on your device using the browser's built-in `chrome.storage.local` API. This data never leaves your browser and is not accessible to anyone other than you.

## Permissions

- **cookies:** Used to read and write cookies on the active tab. No cookie data is sent to any external server.
- **activeTab:** Used to access the current tab's URL to correctly scope cookie operations.
- **storage:** Used to persist saved credential profiles locally in the browser.
- **host_permissions (`<all_urls>`):** Required to read and write cookies across any domain the user chooses to work with.
